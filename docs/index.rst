openspace_rvdata
===================================

**openspace_rvdata** is a Python library for generating OpenSpace assets from research cruise data on the Rolling Deck to Repository (R2R) database at www.rvdata.us.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
